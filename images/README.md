- **hero** – Homepage and page banner images (e.g. tule-springs-park.jpg, tule-springs-hero.jpg)
- **agents** – Agent headshots (e.g. dr-jan-duffy.jpg)
- **villages** – Village photos (northwest, northeast, south, southeast, tule-springs-aerial.jpg)
- **amenities** – Parks, trails, lifestyle (school.jpg, school-kids.jpg, lifestyle.jpg)
- **listings** – Listing photos
- **testimonials** – Client photos for testimonials
- **icons** – Logos, equal-housing.svg, realtor-logo.svg, favicon
